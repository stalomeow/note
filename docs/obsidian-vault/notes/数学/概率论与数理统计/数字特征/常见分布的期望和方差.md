# 常见分布的期望和方差

| 分布    | [[数学期望]]             | [[方差]]                 |
| :---- | :------------------- | :--------------------- |
| 0-1分布 | $p$                  | $p(1-p)$               |
| 二项分布  | $np$                 | $np(1-p)$              |
| 泊松分布  | $\lambda$            | $\lambda$              |
| 均匀分布  | $\dfrac{a+b}{2}$     | $\dfrac{(b-a)^2}{12}$  |
| 指数分布  | $\dfrac{1}{\lambda}$ | $\dfrac{1}{\lambda^2}$ |
| 正态分布  | $\mu$                | $\sigma^2$             |
| 卡方分布  | $n$                  | $2n$                   |
| t 分布  | $0$                  | $\dfrac{n}{n-2}$       |
