# Windows 快捷键

| 快捷键            | 功能    |
| :------------- | :---- |
| Alt+Tab        | 切换应用  |
| Ctrl+Shift+Esc | 任务管理器 |
| Win+L          | 锁屏    |
| Win+D          | 显示桌面  |
| Win+I          | 打开设置  |
| Win+Tab        | 切换桌面  |
| Win+R          | 运行    |
