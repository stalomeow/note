# 梯度（Gradient）

设 $f$ 为一个标量场。

梯度 $\mathbf{grad} \ f$ 是一个向量。它的方向是该点处 $f$ 的最大增长方向。它的大小是这个方向的增长率，是该点处所有方向导数的最大值。

在三维直角坐标系中

$$
\mathbf{grad} \ f = \nabla f = \left (\frac{\partial f}{\partial x}, \  \frac{\partial f}{\partial y}, \  \frac{\partial f}{\partial z} \right)^T
$$
